  Navigateur → DeepL API (❌ bloqué par CORS)

ARCHITECTURE APRÈS :
  Navigateur → /api/translate (Vercel) → DeepL API (✅ fonctionne)

  ✅ api/translate.ts (Vercel Serverless Function, 162 lignes)
     - Runtime : edge (rapide, gratuit)
     - Accepte POST avec body {text, target_lang, source_lang}
     - Détecte auto endpoint free/pro selon suffixe ":fx" de la clé
     - Gestion erreurs : 403 (clé invalide), 429 (quota), 456 (quota mensuel)
