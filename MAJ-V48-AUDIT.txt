# N1 — Prix VIP négocié peut DÉPASSER prix public 🔴 CONFIRMÉ
─────────────────────────────────────────────────
Statut spec V47-BIS : déjà signalé, CONFIRMÉ par audit.

Fichier:ligne :
  `src/front/pages/espace-partenaire/GestionDevisPartner.tsx:94-107` (handleSendVIP)
  `src/front/pages/espace-partenaire/GestionDevisPartner.tsx:246-250` (UI fallback)

Symptôme :
  Partenaire saisit `prix_vip_negocie` qui dépasse `prix_unitaire` (prix
  public). La validation actuelle vérifie SEULEMENT les bornes basées sur
  `prix_achat * coefficient_min/max`, jamais le cap au prix public.

  Ex : prix_achat=100, prix_unitaire=150, coef_max=2.0 → max=200
  Partenaire saisit 195 € → accepté (< 200) → client paie 195 € au lieu de 150 €.
  → Client paie PLUS CHER en VIP qu'en standard. Perte de marge + confiance.

Cause racine (ligne 94-106 actuelle) :
   ```ts
   if (prixAchat > 0) {
     const prixMin = prixAchat * coefs.coefficient_vip_min;
     const prixMax = prixAchat * coefs.coefficient_vip_max;
     if (prixNegocie < prixMin) errors.push(...);
     if (prixNegocie > prixMax) errors.push(...);
     // ❌ MANQUE : if (prixNegocie > ligne.prix_unitaire) ...
   }
   ```

Fix prêt à coder :
   ```diff
       if (prixAchat > 0) {
         const prixMin = prixAchat * coefs.coefficient_vip_min;
         const prixMax = prixAchat * coefs.coefficient_vip_max;
   +     const prixPublic = ligne.prix_unitaire ?? 0;
         if (prixNegocie < prixMin) errors.push(`${ligne.nom_fr} : prix ${prixNegocie}€ < min ${prixMin.toFixed(2)}€`);
         if (prixNegocie > prixMax) errors.push(`${ligne.nom_fr} : prix ${prixNegocie}€ > max ${prixMax.toFixed(2)}€`);
   +     if (prixPublic > 0 && prixNegocie > prixPublic + 0.01) {
   +       errors.push(`${ligne.nom_fr} : prix VIP ${prixNegocie}€ > prix public ${prixPublic}€ — un client VIP ne peut PAS payer plus cher qu'un client standard.`);
   +       hasError = true;
   +     }
       }
   ```

Et ligne 246-250 (UI helper text fallback du max affiché à l'utilisateur) :
   ```diff
   - const prixMax = prixAchat > 0 ? parseFloat((prixAchat * coefs.coefficient_vip_max).toFixed(2)) : (l.prix_unitaire ?? 0);
   + const prixPublic = l.prix_unitaire ?? 0;
   + const prixMaxCoef = prixAchat > 0 ? parseFloat((prixAchat * coefs.coefficient_vip_max).toFixed(2)) : prixPublic;
   + const prixMax = Math.min(prixMaxCoef, prixPublic);  // jamais au-dessus du public
   ```

Sévérité : 🔴 critique (perte CA + confiance client)
Effort : 15 min
Dépendances : aucune
