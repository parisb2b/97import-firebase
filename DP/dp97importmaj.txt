✅ APRÈS (l.105-130) :
```tsx
  const handleAddCustom = async () => {
    if (!customNom.trim()) return;
    const id = `custom_${Date.now()}`;
    let photoUrl: string | undefined;
    if (customPhoto) {
      setPhotoUploading(true);
      try {
        const path = `custom_products/${id}.${customPhoto.name.split('.').pop()}`;
        await uploadBytes(storageRef(storage, path), customPhoto);
        photoUrl = await getDownloadURL(storageRef(storage, path));
      } finally { setPhotoUploading(false); }
    }
    const newItem: CartItem = {
      id, ref: 'SUR-MESURE', nom_fr: customNom.trim(), prix: 0, qte: customQte,
      type: 'custom', description: customDesc, lien: customLien, photoUrl,
    };
    saveCart([...cart, newItem]);
  };
```

### 🗓️ Tâche 2 — N°35-36 : Robustesse anti-flottant cascade Commission

**Pourquoi ça pouvait échouer :** Le calcul `totalHt - totalEncaisse` pouvait produire un float résiduel (ex: `0.0000000001`) empêchant `Math.abs(soldeRestant) < 0.01` d'être vrai. Les montants d'acomptes non normalisés aggravaient le risque.

```tsx
      const totalHt = devis.total_ht || devis.total || 0;
      const soldeRestant = totalHt - totalEncaisse;
```

✅ APRÈS :
```tsx
      const totalEncaisse = Number(acomptesActuels
        .filter((a: any) => a.encaisse === true)
```
