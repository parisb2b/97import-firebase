RÉSUMÉ
------
Suite et finalisation de v35e : implémentation des 4 générateurs Excel restants
(BD-INVOICE, BD-PACKINGLIST, BD-MARITIME, BE-EXPORT), création du modal de
sélection client multi-client, et intégration complète dans DetailConteneur.

Tous les documents sont bilingues FR/ZH avec fallback "À compléter" en italique
gris pour les champs vides, suivant le pattern établi par BC-CHINE (v35e).

WORKFLOW UTILISATEUR (BACK-OFFICE)
-----------------------------------

1. Admin ouvre /admin/conteneurs/:id (DetailConteneur)
2. Section "Documents Excel bilingues FR/ZH" affiche 4 boutons
3. Clic sur un bouton → PopupSelectionClientExcel s'ouvre
4. Modal liste tous les clients ayant des produits dans le conteneur
5. Admin sélectionne le client → clic "Confirmer"
6. Génération du document Excel pour ce client
7. Téléchargement automatique du fichier .xlsx
8. Nom fichier : BD-INVOICE_CTN-260001_ClientNom_2026-04-20.xlsx

DÉPENDANCES FIRESTORE
----------------------

Collections utilisées :
- conteneurs/ : infos conteneur, frais_maritimes, devis_lies
- listes_achat/ : lignes produits avec conteneur_assigne, client_nom, devis_id
- products/ : données produits (nom_zh, nom_en, marque, dimensions, poids, etc.)
- quotes/ : infos client (client_nom, client_adresse, client_siret, etc.)
- admin_params/taux_rmb : taux de change EUR → CNY

Champs clés :
- ligne.conteneur_assigne : filtre les lignes d'un conteneur
- ligne.client_nom : groupe par client
- ligne.devis_id : lien vers quotes/ pour infos client

ARCHITECTURE TECHNIQUE
----------------------

Pattern de génération Excel (commun aux 5 générateurs) :
1. Charger conteneur + lignes avec buildLignesCtn() ou directement
2. Enrichir les lignes avec enrichirLigne() (si besoin)
3. Créer workbook ExcelJS + worksheet
4. Définir largeurs colonnes
5. Titre + en-tête LUXENT (BD-*) ou info conteneur (BC-*, BE-*)
6. Bloc client (si applicable)
7. En-têtes tableau avec styleOrangeHeader() ou styleNavyHeader()
8. Lignes de données avec styleData()
9. Ligne TOTAL avec styleTotal()
10. Conversions CNY ↔ EUR (si applicable)
11. Notes / mentions légales
12. Générer buffer : workbook.xlsx.writeBuffer()

Helpers réutilisables :
- firestoreHelpers.ts : getTauxRmb, enrichirLigne, getInfosClientDevis
- luxentHeader.ts : insertLuxentHeader (BD-* seulement)
- buildLignesCtn.ts : agrégation lignes par client
- excelTypes.ts : strVal, numVal, formatIdExcel, todayFr, downloadExcel
- excelStyles.ts : styleNavyHeader, styleOrangeHeader, styleData, styleTotal
