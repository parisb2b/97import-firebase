=== MAJ35C-FIX1 — Fix tri colonnes ===
Date : 17/04/2026

PROBLÈME :
  Tri cliquable sur en-têtes colonnes ne fonctionnait pas
  (icônes affichées, onClick déclenché, mais ordre des lignes inchangé)

CAUSE :
  Tri calculé via [...filtered].sort() directement dans le render
  React ne détectait pas le changement → pas de re-render

SOLUTION :
  Utilisation de useMemo avec dépendances [filtered, sortCol, sortDir]
  Robustification de la comparaison (undefined, null, Timestamp, string)

FICHIERS MODIFIÉS :
  ✅ src/admin/pages/ListeConteneurs.tsx
  ✅ src/front/pages/espace-client/MesFactures.tsx
  ✅ src/front/pages/espace-client/MesVirements.tsx

   - Import useMemo ajouté
   - Filtrage enveloppé dans useMemo([conteneurs, statutFilter, searchTerm])
   - Tri enveloppé dans useMemo([filtered, sortCol, sortDir])
   - Robustification : gestion Timestamp Firestore + undefined/null
   - else if pour éviter double conversion string

2. MesFactures.tsx :
   - Import useMemo ajouté
